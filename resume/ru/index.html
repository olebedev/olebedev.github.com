ООО "Эквит"

C апреля 2011 по настоящее время, разработчик.

Основные обязанности - создать отпросник в стиле твиттер, с закрытыми вопросами разных типов(текстовые, с изображениями, слайдером, рейтингом). Для нахождения единомышленников. Полная i18n, включая сущности вопросов/ответов/тегов. Под платформу Google App Engine(python/tipfy). После 3х месяцев работы, пришлось полностью переписать систему с фреймворка tipfy на flask и отказаться от GAE в силу экономических причин и того что статус tipfy - deprecated. Как следствие возникла необходимость в архитектуре.

Планируемая архитектура приложения:

- асинхронно реплицируемая база данных на отдельном кластере, обращение по внутренним IP
- несколько application серверов, написанных на python, через wsgi шлюз,
  сброс задач в очередь, никакой статики
- сервер очередей задач, обрашение через внутренние IP
- CDN для статического контента
- большая часть рендеринга вынесена на клиента(браузер)
- comet-сервер для real-time фич

Все поставленные задачи были реализованы, в альфа версии. После релиза альфа-версии 13 сентября, работа над проектом почти остановилась, так как из проекта ушел лидер и идеолог.
